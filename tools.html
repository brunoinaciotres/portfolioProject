Tools Used

Javascript

JavaScript é uma linguagem de programação utilizada principalmente para adicionar interatividade e comportamento dinâmico a páginas da web. É uma linguagem client-side, o que significa que ela é executada no navegador do usuário e pode ser usada para manipular o Document Object Model (DOM) de uma página da web, lidar com eventos do usuário como cliques e teclas, fazer solicitações HTTP para servidores e muito mais.

HTML

HTML (Hypertext Markup Language) é a linguagem padrão usada para criar páginas da web. É uma linguagem de marcação que usa tags para descrever a estrutura e o conteúdo da página da web. As tags HTML são usadas para definir diferentes tipos de elementos na página, como cabeçalhos, parágrafos, imagens, links, formulários e muito mais. A estrutura da página é definida usando uma hierarquia de elementos aninhados, onde os elementos pai contêm elementos filhos.

CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para definir a aparência e o layout de páginas da web escritas em HTML. Com o CSS, é possível controlar o estilo de diversos elementos HTML, como fontes, cores, tamanhos, espaçamentos, bordas, imagens de fundo e muito mais. O CSS permite separar a apresentação visual da página do seu conteúdo, tornando mais fácil e organizado o desenvolvimento e manutenção de uma página web. O CSS usa seletores para escolher quais elementos HTML serão estilizados e regras para especificar como eles serão estilizados. É possível definir estilos para um elemento específico, um grupo de elementos ou para toda a página.
